# Installation on a Fresh Macbook

1. Install git and homebrew
2. Clone repo and run ./dev-setup.sh
3. Restart machine
4. Setup Raycast hotkey to Cmd+Space
5. Launch Karabiner-Elements and enable the permissions there
6. (Setup Yabai and disable System Integrity Protection)[https://github.com/koekeishiya/yabai/wiki/Disabling-System-Integrity-Protection]
7. (Setup Yabai's Scripting Add-On)[https://github.com/koekeishiya/yabai/wiki/Installing-yabai-(latest-release)#configure-scripting-addition]
